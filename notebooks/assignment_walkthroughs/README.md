# Assignment Walkthrough Notebooks

These notebooks give Teaching Fellows a student-facing structure for walking through Assignment 2 and Assignment 3 concepts during office hours, review sessions, or recitations.

They are not answer keys. They include setup cells, concept checkpoints, TODO blocks, debugging prompts, and short reflection questions so students can practice the ideas before writing final assignment answers.

## Files

- `assignment_02_simulation_walkthrough_student.ipynb`: Bernoulli trials, best-of-seven simulation, repeated experiments, histograms, and interpreting simulated probabilities.
- `assignment_03_data_analysis_clustering_walkthrough_student.ipynb`: motorcycle dataset loading, cleaning, filtering, correlation checks, visualizations, and K-means clustering.

## TA-Only Version

The local folder `notebooks/ta_assignment_walkthroughs/` is ignored by git so TA facilitation notes and answer checkpoints do not enter the public repository.
